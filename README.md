🦦 Tyler's Cards
A Singapore-themed interactive flashcard app built for my son Tyler (16 months) to learn animals, foods, colours, and shapes.
🔗 Live App: lujiale93.github.io/tylers-cards

What it does

🔊 Speaks every word aloud automatically when a card appears
🇸🇬 Singapore flavour — local facts on every card (Bishan otters, Gardens by the Bay, kaya toast...)
📱 Toddler-friendly — big tap targets, swipe to navigate, works on phone/tablet
🎯 32 cards across 4 categories — mix and match before each session

Tech
Pure HTML, CSS, and JavaScript — no frameworks, no dependencies. Uses the browser's built-in Web Speech API for text-to-speech.
